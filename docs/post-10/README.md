# 项目介绍

一款基于Vite和Vue3编写的轻量级博客框架，旨在使用Markdown这门简介的文档语言构建和编写博客文章。

# 安装依赖

```bash
npm install

cd scripts

npm install
```

# 启动项目

```
npm run dev
```

# 打包项目

```bash
npm run build
```
