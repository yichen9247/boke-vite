# 写在前面

你有想过进一步美化你的网站或博客吗，当你想更换网站字体却不知道如何更换或者懒于写 CSS 代码的时候，请往下看。

# 插件概述

WordPress 字体插件一款针对 WordPress 站点开发的一款可以替换网站前端字体的插件，利用该插件，站长可以快速实现替换站点前端字体等。

与织音插件不同的是，WordPress 字体插件 具有更美观的设置界面和更全面的字体库，目前字体库已收录字体 35+。
